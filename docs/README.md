# geometric_lib
## Библиотека для подсчётов площадей и периметров некоторых геометрических фигур, а именно Круга, Прямоугольника и Квадрата.

# Параметры считаются по следующим формулам:
## Площадь:
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Периметр:
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
